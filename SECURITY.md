# Security Policy

## Supported Versions

This repository hosts a static website. Security updates and patches are only applied to the current live version of the site (the `main` branch).

| Version | Supported          |
| ------- | ------------------ |
| Live / `main` branch | :white_check_mark: |
| Older commits | :x: |

## Reporting a Vulnerability

We take the security of our website and our users seriously. If you discover a security vulnerability (such as exposed credentials, broken links, or malicious code), please do **not** create a public GitHub issue. 

Instead, please report it to us privately so we can fix it before it is exploited.

**How to report:**
Please send an email directly to: **vidyavantage@yahoo.com**

**What to include in your email:**
* A clear description of the vulnerability.
* Where exactly it is located (URLs or file paths).
* Steps to reproduce the issue.

**What to expect:**
* We will acknowledge receiving your report within 48 hours.
* We will investigate the issue and apply a fix as quickly as possible.

Thank you for helping keep our platform safe and secure!
